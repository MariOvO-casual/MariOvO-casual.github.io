Project Overview

In this programing assignment, user can control the Cylinder and make it to jump from one platform to the next platform.

User Guide

    Jump Control:

  • Pressing the space bar to charge the Cylinder, the longer you hold space bar, farther it can jump.
  • If the cylinder land on the next platform, you can keep jumping. Otherwise, try again.
  • Each time you successfully landed, you gain one point. Press "s" to show the current score.
  • The distance between platforms will keep increasing.
  • You can select whether the static landing platform or moving platform by checking the box above the canvas.

    • Camera Control:

  • Using mouse to control the camera.
  • Hint: Using console to check how long you have pressed the space bar.
